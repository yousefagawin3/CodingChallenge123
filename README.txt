
Persistence:
-Sharepref was used to store the date value in String format in the app.
-Room Database was used to store the List of Songs from the URL

Architecture:
-Model-View-ViewModel was used in the app

Others:
-Coroutines was used for threading, material components where used in the recyclerview, cardview, etc., and Dagger2 was used as its dependency injection, just in case the app will be needing to be modularized in the future.
